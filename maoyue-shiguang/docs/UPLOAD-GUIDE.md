## 四、云开发开通与部署（你本人做，关键）

1. [ ] 微信开发者工具 → 顶部「云开发」→ 开通，创建一个环境，记下**环境 ID**。
2. [ ] 把环境 ID 填到 `app.js` 顶部的 `CLOUD_ENV`（留空则用默认环境）。
3. [ ] 云开发控制台 →「数据库」→ 新建集合 **`couples`** 与 **`users`**（权限设"仅管理端可读写"即可，只经云函数访问）。
4. [ ] 右键 `cloudfunctions/secCheck`、`cloudfunctions/coupleData`、`cloudfunctions/pay` →「上传并部署：云端安装依赖」。
5. [ ] 测试内容安全：发一条含敏感词的纸条/愿望应被拦截。
6. [ ] 测试双向同步：两台手机互相绑定邀请码后，一方加愿望/记账/记心情，另一方约 10s 内或回前台能看到（详见 `docs/CLOUD-SYNC.md`）。
7. [ ] 微信支付（会员收费才需要）：开通商户号 → 云开发后台绑定 → 填 `pay/index.js` 的 `PAY_CONFIG` 与 `utils/pay.js` 的 `PAY_ENABLED`（详见 `docs/PAYMENT.md`）。不收费可跳过，会员走模拟开通。
   - 说明：未开通云开发时，内容安全默认放行、同步自动跳过，开发者工具里照常体验。
