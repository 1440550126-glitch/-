# 05 · 分集走向与伏笔(第一季 12 集)

> 标记:🌱 埋(伏笔) · 🔁 养(深化 / 回扣) · 💥 爆(揭示 / 反转)

## 铁律(全程不可违反,违反即穿帮)

1. **配方与免疫体永远是两个人**:真配方=数据(Ada 前半 + Crane 后半);免疫=Cole 的肉身。Cole 从无完整数据,Crane 从无免疫之身。
2. **两套血清**:公开版=Crane 篡改版(烧脑造怪);真版=原始版(7 号舱,几百人只活 Cole 一人=免疫)。
3. **Crane 动机**:蓄意篡改、放出怪物瘟疫 → 社会崩溃 → 迦南成唯一避难所 → 他当神。真配方是他独占的造神底牌。
4. **终解药 = 完整真配方 × Cole 生物学**。缺任一不成(防穿帮见末节)。
5. **义体退化铁律**:每拷贝 / 转移一次保真度下降;不怕"死",怕"被抹掉"。
6. **续族铁律**:吃脑进化、断食退潮;怕水;脑=安全的水。

## 三幕 12 集走向

### 第一幕 · 荒野 / 组团 / 护人(1–3)

1. **冷开场:不许哭**(Lily)。干尸 2.0 登场——破天灵盖吃脑、清水泼上去尸身腐烂(立新规则)。Cole 流红血暴露被追,Ada 相认塞半截数据。主角团雏形(Cole+Ada+Lily,Bex 救场)。Marek 来抢人。🌱**Lily 在尸群中竟没被攻击**。
2. Cole 被擒放血 → 血让干尸短暂"退烧"现人形=**真解药存在**,离体失效。团队为救他第一次为彼此拼命(**护人主题落地**)。Pip 带路登场。🌱**一只干尸开口说了话**(清醒者惊鸿一瞥)。
3. 黑市换壳铺:Walt 上传翻车"我是哪个我"=🌱**义体退化**伏笔;Ada 读出数据只有一半。Hale 加入。水荒,众人定向迦南。🌱**Hale 瞥见一只清醒尸,脸色骤变**。

### 第二幕 · 逼近迦南 / 三方浮现 / 下水道(4–7)

4. **隔江看迦南**(阶级鸿沟 + 水即权力)。Marek 要拿 Cole / Lily 当门票。发现墙上进不去——**墙下有下水道**。🌱下水道口贴着"地火"的记号。
5. 钻进下水道,撞见**地下组织 · Silas**。他要数据、要 Cole,口号"推墙放水"像义军,手段却狠。Cole 险失控(免疫代价揭示)。🌱**Pip 和 Silas 的人眼神交流**(内鬼)。
6. **续族登场:Vivian** 截下主角团却不杀,坐下"谈判"——她要 Lily。揭示干尸会进化、有社会、有计划(反常识:和怪物谈判)。Hale 当场认出旧同事 Vivian。🌱**Vivian 说"那针是假的,去问 Crane"**(她知道真相)。〔完整分镜见 `剧本/第06集_和怪物谈判.md`;脑库农场见 `剧本/场景_脑库农场.md`〕
7. **三方拼图**:7 号舱最高权限仅 3 人=**Crane、Vivian、Silas**(三阵营头目都知真相!)。Ada 的秘密爆一半(她早认得他们)。线索:锁死 Ada 权限、篡改数据的是同一只手=Crane。🌱**Bex 的渴血期被 Cole 撞破**。

### 第三幕 · 迦南内 / 四方收口 / 解药(8–12)

8. 经下水道潜入迦南。物化压迫真相;**Lily 被四方同时盯上**。见到"英雄医生"Crane(藏在明处)。💥**Pip 内鬼曝光**——团队因内部裂痕险些全灭。
9. **反转集**:摆渡人=Crane=数据贼=推 Cole 进舱的人=三权限者之一;自曝"七个备份,杀不死我"。同时 **Vivian 率续族经下水道灌城**。三线收口、四方汇于下水道。
10. **归墟 · 江远登场**。两把钥匙:Crane(后半配方)× Cole(活体);新变量——解药对清醒者=灭种,Vivian 死战;Silas 想趁乱夺城。💥**Lily 的秘密揭晓:天生免疫**(钥匙二的保险),四方抢她到顶点。
11. **下水道总决战**(冲突顶点):主角团为保 Lily 与彼此死战(护人高潮)。💥**Bex 在此刻彻底转化,用最后一丝清醒替团队挡住尸潮**(悲情爆点)。江远拆穿 Crane 备份退化("你正被抹掉")逼其交出后半配方。Silas 欲放水淹城连平民牺牲,被主角团拦下,**Pip 反水救赎**。
12. **季终**:Ada 半截 + Crane 后半=完整真配方,× Cole & Lily 生物学 → 归墟**量产免疫因子,免费投放**。注射波及清醒者——给 **Vivian 一个有重量的结局**(被"治愈"回人 / 主动赴消亡)。迦南墙塌、水放,被物化者获自由与公平。Lily 终于能哭,**没人再捂她的嘴**;枯地冒新绿。Cole 从独狼,成了一群人的家。
