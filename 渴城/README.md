## 一句话故事

水即人性的末世。辐射让人不断失水,彻底脱水的人会变成专吃活人脑髓的「**干尸**」;而吃够聪明脑子的干尸会**进化、会思考**,自成一族。一座实验室几百名受试者中,只有 **Cole** 一人对辐射免疫——他的身体是解药的一半;**另一半,是被一名医生偷走并篡改的真实实验数据**。随着水源枯竭,幸存者、义体永生的权贵、进化的尸族、藏在下水道的地下组织,全都向背靠江岸的「**迦南**」未来城涌去。在这个没有法律、只有自由与暴力的时代,Cole 和他的伙伴必须学会一件最难的事——**保护身边的人**。

## 三大记忆点(引流卖点)

1. **水的反转**:在人人渴死的世界,**水是杀干尸的武器**;眼泪是奢侈品,**哭是死罪**。
2. **怪物的反转**:干尸不要你的血,**要你的脑**;最聪明的"人",可能是个死人。
3. **两把钥匙**:制造末世的人,正是手握唯一解药配方的人;**偷数据的人 ≠ 免疫的人**,缺一不可。

## 基调与内容边界

- 末世压迫(迦南权贵对城外人的奴役、物化)一律以「**把人当作财产 / 商品 / 消耗品**」的**物化恐怖**呈现,不写成情色或性暴力画面。
- 这与全剧弧光一致:**压迫 → 反抗 → 解放**;结局指向被压迫者(含女性)获得自由与公平对待。
- 暴力 / 恐怖服务于惊悚与主题,留白与压迫感优先于血浆,利于过审与传播。
