# 01 · 世界观与设定

## 一、永生计划(EVERLIFE)——一切的源头

富人与政府为躲避辐射,启动「永生计划」,分**血**与**铁**两条线:

| 线 | 内容 | 结果 |
|---|---|---|
| **甲线 · 血** | 抗辐射血清(肉体永生) | **失败**。几百名受试者只活下一个——**Cole**,且唯一免疫。后被医生篡改数据,公开版血清反成"造怪物"的瘟疫源头。 |
| **乙线 · 铁** | 意识上传义体(数字永生) | **成功**。但贵到只有顶级富人买得起 → **义体人**的由来。 |

- **真血清 vs 篡改血清**:真版=原始版(7 号舱试验,几百人只活 Cole 一个=免疫);公开版=被 Crane 篡改版(烧脑造怪)。**Cole 是"真配方有效"的唯一活证据。**

## 二、失水三阶段

辐射让人体不断失水:

1. **燥化期**:暴躁、易怒、皮肤皲裂出血。
2. **渴血期**:理智崩塌,靠喝活人血补水续命(血是唯一还能"骗过"身体的水)。
3. **干尸化**:彻底脱水后,不再是人,而是**会攻击活人、专吃脑子**的怪物。

## 三、干尸 2.0(去酸血 · 加吃脑 · 加进化)

- 干尸:不流血、不怕痛、力大,专破天灵盖**吃脑**——脑是干涸世界里水分与脂质最丰的器官。
- **进化(全剧最大反常识)**:吃够新鲜、聪明的脑 → 恢复思考、记忆、语言,**越吃越聪明** = 「清醒者」 = 干尸精英与首领;它们自称「**续族**」(详见 `04_续族设定集.md`)。
- **弱点 = 水**:纯水让脱水尸身迅速腐烂剥落——**在渴死的世界,水是杀干尸的武器**(故水=绝对权力)。
- **判定规则**:受伤**流红血 = 还是人**;不流血 / 渗清液 = 正在变 / 已变。

## 四、义体人——需要充电的有钱人

- 富人花天价把记忆意识上传进义体,**从此不喝水、不会渴、不会尸化,只需充电**——直接退出"人类"这个物种。
- **黑市拼装**:穷人买不起正规手术,淘报废部件**自己拼装**,结果拷贝出错、记忆错乱、半机械半尸化的残品满街。
- **备份退化(伏笔机制)**:每拷贝 / 转移一次,保真度下降;备份越多 = 代数越高 = **自我流失越重**(拷贝会问"我是哪个我",问多了自我烧毁)。

## 五、迦南(未来城)——江对岸的应许之地

- 政府与富人联手,在大江边建城:**取之不尽的水、最顶级实验室与团队、义体永生**——但**平民不得入内**。
- 城内:喷泉当街喷涌(渴死世界里的炫耀)、绿树、灯火、义体永生者;**公民全用旧世界西式名字**,自称"留住文明最后一批名字的人"。
- 城外:渴城、血站、干尸、军阀。
- 压迫真相:迦南把城外人**分级、烙印、当财产交易**(物化恐怖,非情色)。

## 六、地理与势能:都在往迦南挤,墙下有下水道

- 水源枯竭 → 所有人向迦南靠拢(**江岸=最后的水,黑市=最后的交易**),地图向内坍缩,矛盾被挤到一处。
- 迦南墙上铜墙铁壁,**墙下=旧城下水道 / 地下城**,连通江与荒野——**唯一缺口,兵家必争**。
- **「再高的墙也有下水道」= 全剧命门**:主角团从这潜入、地下组织住在这、续族要从这灌进城,终局四方在下水道收口。
- **水的攻防**:续族怕水,故迦南的江水=天然护城河;续族只能走**干燥的下水道**进城——这正是下水道成为命门的物理原因。

## 七、统一母题:为了不消失,不断付代价

三条求生路殊途同归:

- **普通人**:囤水 → 失水就变尸。
- **迦南富人**:义体拷贝 → 每拷一次丢一点。
- **续族**:吃脑 → 断食就退潮(变回无脑游尸)。

三者都在"**为了不消失,不断付代价**"——这是全剧的统一母题。

## 八、反常识规则集(钩子库)

- 哭是死罪——流泪=暴露你还有水、也=在浪费救命的水。
- 受伤流红血是好事,证明你还是人;伤口冒白烟 / 流清液,说明你在"变"。
- 杀干尸不用刀枪,用**水**;一场雨能淹掉一支尸潮。
- 干尸不要你的血,要你的**脑**。
- 末世最聪明的"人",可能是个死人(清醒者比活人会算)。
- 末世里最安全、最滋润的地方,可能是当续族的"口粮"(见脑库农场)。
