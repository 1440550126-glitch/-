# 03 · 主角团

> 主题:在没有法律、只有自由与暴力的乱世,**学会保护身边的人**。
> 主角团是一个"找到的家人"(found family);核心情感引擎是他们彼此守护(尤其守护 Lily),而 Cole 的弧光是从"独狼"成长为"一群人的家"。

| 角色 | 性格 | 想要 | 秘密 / 伏笔 🌱 |
|---|---|---|---|
| **Cole**(免疫男主)【钥匙二·活体】 | 沉默克制的独狼 | 别拖累谁,独活 | 免疫有代价——他在**缓慢逆向变异**(越近尸化越清醒冷),怕终有一天伤到队友。弧光:独狼 → 护家人 |
| **Ada**(女主,前数据员)【钥匙一·半数据】 | 聪明、被内疚驱动 | 翻案、护住 Lily | 当年**她亲手封了 Cole 的舱、只抢出半截数据**;她**认得 Crane、Vivian、Silas**(旧上司)——比谁都早知道三阵营头目是谁 |
| **Lily**(不许哭的女孩) | 团队的心 | 不再害怕 | "不怎么渴"、脑子"不一样"——疑似**天生免疫的第二人**;**四方都想要她**(Crane 要样本、Vivian 要免疫脑、Silas 要筹码) |
| **Bex**(退役女兵,肌肉) | 凶悍、毒舌、死忠 | 像护战友一样护住团队 | **已进渴血期、在偷偷变**,瞒着所有人跟时间赛跑——定时炸弹 + 悲情 |
| **Pip**(少年拾荒 / 黑客) | 机灵、贫嘴、灰色 | 活下去、找归属 | **在给地下组织(Silas)递消息**——内鬼 / 双重忠诚,迟早爆;后反水救赎 |
| **Hale**(老医护,良心) | 温厚、有智慧 | 守住人性底线 | **认得干尸首领 Vivian(旧同事)**——成为通往续族阵营的活桥 |

## 关系网与张力

- **Cole ↔ Lily**:守护关系。Lily 是 Cole"学会保护别人"的支点。
- **Cole ↔ Ada**:互相隐瞒的两个人(他瞒变异,她瞒身世与内疚),信任在揭秘中艰难建立。
- **Bex 的倒计时**:她在变,知道自己迟早要被队友"处理"或自我了断,所以拼命在最后的时间里护住所有人 → 第 11 集转化挡尸潮的悲情爆点。
- **Pip 的双面**:他递消息给 Silas,危机时几乎害死全队(第 8 集曝光),最终在第 11 集反水救赎——是"护人主题"的反面与回归。
- **Hale 的桥**:他与 Vivian 的旧识身份,让"与怪物谈判"有了人性的重量,也带出真相线。

## 弧光收束(季终)

Cole 从"我谁也不护"到为守护众人付出代价;主角团以惨烈代价(Bex 牺牲)守住 Lily 与彼此。Lily 终于被允许哭泣——**没人再捂她的嘴**——是"自由与公平回归"的情感落点。
