# 04 · 干尸阵营设定集:「续族」

> 他称:干尸 / 游尸 / 清醒者。自称:**续族**(承嗣者 / 继续活下去的人)。
> 定性:人类造出它们(Crane 篡改的血清)又抛弃它们。在续族眼里,它们不是瘟疫,而是**人类本该进化成的样子**——不靠喝水、靠"吃脑"活下去,并且**让任何人都不会真正死去**的新物种。

## 一、核心机制(三条反常识铁律)

1. **进化 = 吃脑;吃脑 = 继承。**
   吃下新鲜、聪明、含水的脑,会恢复思考、记忆、语言,并**继承被吃者的记忆、技艺、甚至嗓音**。高阶续族是它吃过的所有人的**拼接体**——会用死者的声音说话,知道死者的秘密。(悬疑富矿:一个续族可能"认识"主角,只因它吃过认识主角的人。)

2. **清醒不是天赋,是债务(饥饿之锁)。**
   借来的心智会褪色。**断食 → 众魂消散 → "退潮"变回无脑游尸。** 越聪明越饿,越高位越怕饿。

3. **脑是"安全的水";水是续族的火。**
   纯水让脱水尸身腐烂(怕水甚于人怕火,一场雨能让尸潮溃成泥);但脑髓里**锁住的水**安全。于是脑同时是它们的**水、知识、货币**——它们的"农业"就是**养脑**。

## 二、清醒度等级(看"湿润度"识阶位)

外观逻辑:**越饱食 = 越"湿润" = 越高阶;一饿就风干掉阶。**

| 阶 | 名称 | 吃了什么 | 外观 | 行为 |
|---|---|---|---|---|
| 0 | **游尸**(Drifting) | 残渣、烂脑 | 枯裂如柴 | 无脑,循"湿脑气味"成群猎杀 |
| 1 | **嗅者**(Stirring) | 几颗脑 | 裂纹变浅 | 兽性狡黠、群体战术、记得一张脸 |
| 2 | **清醒者**(Lucid) | 够多的新鲜聪明脑 | 半潮润,暗处近人 | 连续思考、说话、说谎、伪装、布局——阵营"军官" |
| 3 | **通晓者**(Manifold) | 大量高质脑 | 几可乱真,眼神不对 | 天才级借来知识技艺;但脑中太挤,**随时被某个死者"夺舍"** |
| 顶 | **始者**(First)= Vivian | 最早觉醒、吃得最多 | 最像活人 | 最清醒,也**最易碎**(脑中装着两百个死人) |

> 反常识:**最强 = 最不稳定**。顶阶随时被一个"声音"接管,力量越大越接近碎裂。

## 三、脑库:它们怎么"养"人(最阴的反常识)

续族要新鲜、聪明、水足的脑,所以**不屠宰,种植**:

- **越舒服的牲口 = 越好的收成。** 供水、喂饱、安顿,因为健康活跃滋润的脑"出产"更高。
  → **反常识王炸**:末世里人类最安全、最滋润的地方,是当续族口粮。城外渴死、城里被物化,**农场里却有干净的水和书**。
- **让牲口思考。** 出谜题、教书、允许相爱成家——情感越丰沛、智识越活跃,脑越值钱。农场像畸形的学校 / 温泉。
- **按智分级。** 学者、工程师、艺术家是珍品(吃了能继承其技艺);损脑喂游尸。→ **迦南=最密集、最有学识、最滋润的脑库=终极粮仓**,这才是续族倾巢扑向迦南的真正原因。
- **选育。** 农场里出生、长大、**不知道自己是牲口**的人。
- **收割 = 继承(它们的宗教)。** 续族不说"杀",说"**继续他**";被吃者活在吃他的人体内,被一代代带下去。"你们让人烂成虚无;我们让没有人真正死去。"
- **货币与统治。** 脑 / 清醒=经济;低阶为高阶服务换取"进食权";**Vivian 配给脑髓 = 配给清醒 = 绝对权力**。

## 四、Vivian 的"新种"理想

- **史观**:"人类造了我们又叫我们怪物。可我们不渴、不怕痛、难死,还带着死者前行。我们是人类本该进化成的、能在无水世界活下去的样子。"
- **她要修复的缺陷**:清醒=债(饥饿之锁)+ 退化。她在找**稳定、自持、不靠不断进食也能清醒**的续族——而**免疫脑(Lily)/ 免疫因子 / Cole 生物学**,可能正是那块拼图。→ 她的科研目标与主角团的"解药"**是同一样东西,目的相反**。
- **辩才(让观众不适)**:"你们叫这吃。我们叫这——记得。"
- **矛盾 = 伏笔**:她最清醒因为吃得最多;脑里装的死人最多,**包括一名 7 号舱旧同事**——她体内"存着"**Crane 篡改数据的第一手记忆**,是一具会走路的**活人证 / 真相档案**(撬 Crane 之外的第二条真相线)。
- **软肋**:配给清醒=暴政;"孩子们"既崇拜又畏惧;**纯掠食派想直接饕餮**,不信新种高论——内政随时炸。
