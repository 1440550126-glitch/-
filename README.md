# AI句灵 · 让每一句话活过来

面向年轻人的 **AI 文案社交圈**：发一句话 → AI 生成专属预览卡 → 文字**活过来**（3D 流体粒子动画 + 实时合成音效）：发帖那一刻文字即在 curl-noise 流体场中汇聚成句、加性辉光全息呈现，长按卡片可全屏回放。配上 AI 暖场官、轻量桌游大厅（谁是卧底 · AI 主持）、皮肤商城与 9.9 元会员的完整商业闭环。

> 产品定位、功能取舍、风险判断见 [docs/PRODUCT.md](docs/PRODUCT.md)（建议先读）。

## 一键运行（零依赖）

只需要 **Node.js ≥ 22.5**（内置 SQLite / fetch），不需要 `npm install`，不需要数据库服务：

```bash
npm start
# 📱 用户端    http://localhost:3000
# 🛠 管理后台  http://localhost:3000/admin   （默认 admin / jvling-admin-2026）
```

默认**不配置任何大模型 Key 也能完整体验**：所有 AI 能力（预览卡 / 动画导演 / 暖场 / 话题 / 主持）自动落到本地规则引擎，成本为 0。接入大模型只需复制 `.env.example` 为 `.env` 填入任意 OpenAI 兼容接口（火山方舟 / 阿里百炼 / 腾讯混元 / DeepSeek）。

```bash
npm run smoke            # 86 项全链路冒烟测试（卧底×2 + 狼人杀完整对局）
node scripts/llm-test.mjs  # 大模型连通性一键自检（配好 .env 后运行）
npm run dev     # 开发模式（文件变更自动重启）
```

## 已实现功能

**文案社交圈**：发布（敏感词三级审核）、推荐/最新/关注/热门四种信息流、点赞、评论与楼中楼回复、收藏、分享、关注、个人主页、今日话题（AI 生成 · 带标识）。

**个性化推荐「越来越懂你」**：从点赞/收藏/评论/关注构建兴趣画像（情绪/作者/话题，按交互强度加权 + 时间衰减），按「亲和度 × 个性化强度 + 质量 + 新鲜 + 探索」排序并做多样性去重；**个性化强度随互动增长——越用越懂你**；**新用户冷启动自动回退到热度+新鲜（保证引新）**；「不感兴趣」负反馈即时降权同类；每条带可解释 `rec_reason`，配「句灵越来越懂你」口味画像横幅，让个性化被用户感知到。

**文字变动画（核心卖点）· v3「未来已来」**：服务端大模型当"导演"产出 Animation Manifest（情绪/场景/粒子/时间轴/声音），客户端引擎做"播放器"——成千上万颗发光粒子在 **curl-noise 流体场**中以**透视景深**汇聚成你写下的句子，**加性辉光（holographic bloom）** 做出全息质感，缓慢摄像机偏航带来 3D 体积与视差。情绪不是标签而是"运动签名"：热血爆发上冲、心动心跳脉冲、思念横向飘散、难过下坠——**文字怎么动，就在表达什么情绪**。彻底告别旧版描边简笔画。配 Web Audio 实时合成音效（风/雨/心跳/碎裂，零音频资源）。**发帖即播放**（纯客户端、零延迟、不耗配额），长按卡片可切风格全屏回放；免费 3 次/天，会员解锁樱粉/霓夜/星海，高级风格走星尘额度。

**AI 暖场系统**：3 个人设账号（小句灵/灵感官/主持官），自动发帖（每日 10-30 条可配）、延迟 30s-10min 评论用户新帖（每帖 ≤2 条）、每日话题、大厅组局提醒（仅大厅无房时）、夜间自动降频、每日成本预算封顶、用户可关闭、后台一键全关。**所有 AI 内容带标识，AI 点赞单独计数不进热度（不伪造热度）**。

**AI 治愈陪聊（句灵陪你说说话）**：1 对 1 倾听陪伴——共情式回应、确认感受、温柔的开放式提问，帮年轻人表达情绪、缓解压力。接 DeepSeek 等任意 OpenAI 兼容接口；**未配置 Key 也可用**（本地共情兜底，按情绪给出温暖回应，零成本不掉线）。每日成本封顶、防刷配额。**安全第一**：内容先过审核，识别到自伤/轻生风险时走**确定性关怀响应**（温柔陪伴 + 心理援助热线 12356，绝不调用模型自由发挥、绝不娱乐化），并记审核日志人工跟进；全程标识「AI 生成」，绝不冒充专业心理咨询。

**桌游大厅**：游戏引擎框架（房间/座位/AI 主持/AI 陪练/计时器为共享底座）+ 三款完整玩法——**谁是卧底**（发词/轮流描述/投票/复盘）、**狼人杀**（狼人/预言家/女巫/平民，天黑行动→白天发言投票完整循环，夜间行动走私有通道、狼队私聊、查验记录、女巫药剂管理）、**迷雾庄园·凶夜**（多人恐怖社交推理：凶手/通灵者/守夜人/幸存者 +（6 人起随机出现的）**第三方小丑**——小丑被投票放逐即第三方获胜，逼得众人不敢乱投；每夜猎杀→感应→守护，天亮抽取**随机灵异事件牌库**制造不确定性与误导，**通灵线索不一定可靠**；撑过 6 夜救援抵达幸存者胜，凶手清场则凶手胜；**身份随机配置+事件随机+线索不可靠+小丑变量 → 局局不同、玩得久不腻**）。SSE 实时同步，AI 陪练补位带明确标识，一个人也能开局；前端用**通用夜间动作协议**声明式渲染，新玩法零前端改动即可接入。剧本杀/密室逃脱排期见 ROADMAP。

**商业化**：9.9 元/月会员（季卡/年卡）、19 款皮肤五档稀有度（卡片边框/头像框/聊天气泡/动画特效/房间主题，**纯外观不影响公平**）、星尘额度（高级 AI 按次扣点、会员 8 折、生成前展示预估消耗）、沙盒支付完整订单流（正式上线替换为微信/支付宝/Apple IAP 回调即可）、青少年模式禁止消费。

**审核与风控**：敏感词三级分类（拦截/转人工/自伤关怀）+ **大模型机审第二道防线**（后台开关、预算封顶、结果通知作者）、自伤内容温柔提示心理援助热线并进入人工审核（绝不娱乐化）、举报全类型覆盖、拉黑、封禁、注销账号匿名化、审核日志、限流防刷（对局动作独立计桶）、每日配额。

**通知中心**：赞/评论/回复/关注/AI 互动/系统审核结果全覆盖，小铃铛实时角标（SSE 推送），一键已读，防轰炸去重。

**管理后台**：数据总览（收入/成本/活跃）+ AI 机审开关、人工审核队列、举报处理、内容/用户管理、AI 暖场配置（频率/夜间/一键开关/手动触发）、AI 成本日报（按天/按功能/逐条）、**桌游房间监控（可强制关闭违规房间）**、皮肤调价上下架、订单、敏感词热更新、操作日志。

## 技术架构

```
┌── web/      移动端 H5 App（原生 ES Modules，零构建）
│     ├ js/anim/   动画引擎 + Web Audio 合成 + 播放器
│     ├ js/fx/     WebGL 流体背景（fbm 噪声 shader）
│     └ js/pages/  12 个页面
├── admin/    管理后台 SPA
├── server/   Node 22 零依赖服务端
│     ├ lib/       db(node:sqlite) / 鉴权 / 审核 / LLM 客户端 / Manifest 导演引擎
│     ├ routes/    55+ REST 接口 + SSE
│     ├ game/      游戏框架 core + 卧底/狼人杀/恐怖(迷雾庄园)三引擎
│     └ warmup/    AI 暖场调度器
├── miniprogram/  微信小程序版（历史版本，可作为渠道端）
└── docs/     产品定位 / API / 数据库 / 合规
```

技术决策的理由（为什么不是 Flutter+NestJS+PG 起步）见 [docs/PRODUCT.md](docs/PRODUCT.md) 第三节；Animation Manifest 协议与端无关，未来 Flutter 客户端按协议实现渲染器即可复用全部服务端。
