# LingMirror AI v1.0 LTS 商用冻结版

LingMirror AI / 灵镜 AI is a Node.js 20 + SQLite commercial AI production platform for overseas AI video scripts, product ads, copywriting, business promos, e-commerce creative assets, knowledge-base assistance, poster planning, 3D asset briefs, memory anchors, AI Director QC and project memory snapshots.

Run locally:

```bash
cp .env.example .env
npm install
npm run migrate
npm run start
```

Open `http://127.0.0.1:3001/index.html`.

Production deployment is documented in `README_FINAL_DEPLOY.md`.
