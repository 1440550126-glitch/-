LingMirror remembers your project.

Every project gets a Visual Bible and Memory Anchor database. After each generated shot, LingMirror extracts frames, analyzes characters, props, scenes and style, saves them as memory anchors, and uses them again before generating the next shot.

每个项目都有视觉圣经和记忆挂点。每生成一个镜头，系统自动抽取首帧、尾帧、关键帧，识别人物、场景、服装、道具和色调，存入项目记忆库。下一个镜头生成前，系统会自动读取记忆并挂到提示词里。
