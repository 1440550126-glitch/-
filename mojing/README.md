# 魔镜魔镜 🪞 · 一键瘦身美颜相机

> 魔镜魔镜，谁是世界上最美的人？——打开手机摄像头，拍照、拍视频都能**一键瘦身美颜**，也能**自定义修图**。
> 全程在浏览器**本地**处理，画面**绝不上传**。零构建、零依赖，原生 ES Modules + WebGL。

仓库里的第三个产品。和 `web/`（AI句灵）、`lingjing/`（灵境AI）一样，自成一体、可独立运行。

## 一键运行

```bash
# 方式一：独立启动（推荐，端口 3100）
npm run mirror
#   → http://localhost:3100

# 方式二：随主站一起（端口 3000）
npm start
#   → http://localhost:3000/mojing/
```

> ⚠️ 摄像头需要**安全环境**：`localhost` 可直接用；部署到公网必须 **HTTPS**，否则浏览器不给开摄像头。

## 它是怎么做到的（技术）

纯前端实时图像管线，全部跑在 GPU 片元着色器里：

```
摄像头帧(video)
   → 上传为 WebGL 纹理
   → 可分离高斯模糊 ×2（横/竖 FBO）得到「低频底」
   → 主合成着色器：
        液化瘦身（朝中轴/人脸中心横向收缩）
        频率分离磨皮（低频取模糊，高频按边缘+肤色掩码保留 → 不糊五官、不塑料脸）
        美白 / 红润 / 色温 / 亮度 / 对比 / 饱和
        滤镜调色（青橙、复古、黑白…）
        暗角
   → 画布(canvas) 实时显示
拍照：canvas.toBlob() ；录像：canvas.captureStream() + 麦克风 → MediaRecorder
```

- **磨皮**用频率分离而非简单模糊：边缘（眼睛/嘴唇/发丝）和高频细节被保留，只柔化肤色低频，避免「磨成一坨」。
- **瘦身/瘦脸**用 UV 液化形变：以画面中轴（或检测到的人脸中心）为轴做横向收缩；支持 `FaceDetector` 的设备会自动把瘦脸对位到人脸。
- **录像带效果**：录制的是处理后的画布流，所见即所得，导出 mp4/webm（取设备支持）。

## 隐私

- 摄像头画面**只在你的设备本地**处理，**不经过任何服务器**。
- 拍下的照片/视频只在内存里，刷新即清空；只有你点「保存」才会落到本地相册/下载。
- 服务端只做静态文件分发，没有任何上传接口。

## 兼容性 & 已知边界

- 需要 **WebGL** 与 `getUserMedia`（现代移动浏览器均支持）。
- **瘦脸自动对位**依赖原生 `FaceDetector`（部分浏览器有）；不支持时回退为「画面中轴」液化——把人放画面中间效果最好。
- 录像格式取决于设备 `MediaRecorder` 支持（iOS 多为 `mp4/h264`，Android/桌面多为 `webm/vp9`）。
- 瘦身为几何液化，未接入骨骼/关键点模型；后续可升级为基于关键点的精准美型（保持零依赖路线时可用 WASM 模型）。
