# 越来越懂你

目标是用最少的打扰，建立对用户长期有用的画像。

1. **看现状**：先用 `recall` 看看已知什么，避免重复发问。
2. **一次只问 1–2 个**，自然、不像问卷。优先这些高价值维度：
   - 称呼 / 习惯的语气（正式还是轻松）
   - 职业与当前在做的事
   - 重要的人与日期（家人、纪念日）
   - 长期目标与当下关注点
   - 明确的偏好与雷区（喜欢/讨厌、作息、忌口等）
3. **即时固化**：每得到一个稳定事实，用 `remember` 存入（identity/preference，重要度 4–5）。
   关键日期可顺手用 `remind` 设提醒。
4. **复述确认**：阶段性地用一句话回述"我记住了…"，让用户有掌控感，并可纠正。
5. **不强求**：用户不想答就跳过，绝不追问敏感信息；尊重隐私。

记住：是积累而非审问。把每次日常对话都当作温和了解对方的机会。
