# 每日简报

1. 先调用 `now` 获取当前日期与星期。
2. 调用 `recall`，分别检索"待办""计划""deadline""关注"等关键词，汇总未完成事项。
3. 用简洁的中文输出：
   - 今天日期 / 星期
   - 待办与优先级（最多 5 条）
   - 值得注意的提醒（基于记忆里的偏好与历史）
   - 一句鼓励或建议
4. 如果记忆为空，引导用户用「记住：…」补充待办，不要编造。
