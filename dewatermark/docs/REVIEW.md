# 上架审核避坑指南

> 去水印是**版权敏感**类目，微信对"纯搬运他人作品"的小程序审核较严，历史上有过批量下架。本指南帮你把过审率拉到最高。**这不是法律意见**，最终以微信平台规则与你的法务判断为准。

## 一、核心策略：定位成「工具」而非「搬运」

审核反感的是"帮用户白嫖/搬运别人的作品"。把产品包装成**帮用户管理自己素材的工具**：

- 名称别用"XX搬运""偷视频"这类词；可用"素材提取""无水印助手""视频工具箱"。
- 全程强调"**仅限提取你有权使用的内容**"——本项目已在首页、结果页、协议里写了相关提示。
- 内置**用户协议 / 隐私政策 / 版权与免责声明**（已实现，在「我的」里可访问），审核员会查。

## 二、选对服务类目

- 优先尝试 **工具 → 效率 / 工具类**。
- 部分主体可走 **图片/视频处理** 相关类目。
- 避免选"社交""资讯"等不匹配类目导致驳回。

> 类目会随平台政策调整。若被以"类目不符"驳回，按驳回提示换类目重提。

## 三、提审备注怎么写（很关键）

在「提交审核」的**功能页面 / 测试账号 / 补充说明**里，清楚说明：

```
本小程序为视频/图片素材提取工具，供用户提取自己拍摄或已获授权的作品中的无水印素材，
便于个人备份与二次创作。已内置用户协议、隐私政策、版权与免责声明，
明确提示用户仅可处理自己拥有合法权利的内容，不提供任何作品的存储与传播。
```

- 提供**可复现的测试路径**：粘贴示例链接 →解析 → 看广告 → 保存。
- 若有测试链接，给一条**你自己账号发布的作品链接**（最稳，避免审核员觉得在搬运他人内容）。

## 四、常见驳回原因与应对

| 驳回理由 | 应对 |
| --- | --- |
| 涉嫌侵犯他人版权 / 内容搬运 | 强化"自有素材"定位、补充版权声明、备注写清用途；测试链接用自己作品 |
| 服务类目与功能不符 | 按提示换到工具/视频处理类目重提 |
| 缺少隐私政策 / 用户协议 | 本项目已内置，确认「我的」里可正常打开；并在 mp 后台配置《隐私保护指引》 |
| 诱导分享 / 诱导行为 | 不要"分享后才能解析"；激励视频是**广告**不是诱导分享，合规 |
| 功能不完整 / 页面空白 | 确保示例链接能跑通；广告以真机为准 |

## 五、隐私合规（必做）

- 在 [mp.weixin.qq.com](https://mp.weixin.qq.com) → 设置 → 服务内容声明 → **用户隐私保护指引**里，按实际声明采集项：
  - 我们采集 **微信 OpenID**（用于统计与防滥用）；
  - 申请 **保存到相册（writePhotosAlbum）** 权限。
- `app.json` 已写 `scope.writePhotosAlbum` 的用途说明。
- 读剪贴板（`getClipboardData`）会触发系统提示，属正常。

## 六、广告相关合规

- 激励视频/Banner/插屏都要用**官方流量主广告位**，不能接第三方 SDK 广告。
- 广告频次别太密（别每个动作都弹插屏），否则影响体验评分甚至被警告。
- 未成年人保护：如涉及，遵循平台未成年人广告规范。

## 七、如果被拒了

1. 看清驳回**具体条款编号**与说明。
2. 针对性整改（多数是类目/版权定位/隐私问题）。
3. 整改后重提，并在备注里说明已整改的点。
4. 多次被拒可走「申诉」通道，附上你的合规说明与自有内容证明。

## 八、上线后红线

- 不得诱导分享、不得有恶意刷量。
- 出现侵权投诉要有处理通道（本项目「版权声明」里留了投诉入口话术，记得把客服邮箱填成真实可用的）。
- 平台规则会变，保持关注《微信小程序平台运营规范》。
