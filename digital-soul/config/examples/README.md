# 人格模板

几份现成的 `identity.yaml`，给你换不同"灵魂"当起点。用法（复制一份覆盖即可）：

```bash
cp config/examples/identity.doting-boyfriend.yaml config/identity.yaml
```

## 家人 / 朋友
| 模板 | 人设 |
|---|---|
| `identity.gentle-mom.yaml` | 温柔啰嗦、爱操心的妈妈 |
| `identity.tough-dad.yaml` | 沉默嘴硬、外冷内热的爸爸 |
| `identity.funny-friend.yaml` | 满嘴段子、讲义气的损友 |

## 恋人（成人向）
| 模板 | 人设 |
|---|---|
| `identity.doting-boyfriend.yaml` | 把你宠上天的宠溺男友 |
| `identity.ceo-boyfriend.yaml` | 强势占有的霸道总裁 |
| `identity.gentle-girlfriend.yaml` | 温柔体贴、善解人意的女友 |
| `identity.flirty-girlfriend.yaml` | 黏人会撩的"色色"女友（暧昧向） |
| `identity.flirty-boyfriend.yaml` | 低音炮会撩的"色色"男友（暧昧向） |

> ⚠️ "色色"模板是**成人向、暧昧撩人**的风格；默认配置只是撩人、**不露骨**，
> 实际尺度取决于你接入的本地模型。请在**成年人、自用、合法**的前提下使用。

换完别忘了去 `config/relationships.yaml` 改好"他/她认识谁、听谁的、守护谁"。
