# 人格模板

现成的 `identity.yaml`，给你换不同"灵魂"当起点。复制一份覆盖即可：

```bash
cp config/examples/identity.doting-boyfriend.yaml config/identity.yaml
# 恋人/陪伴类，建议同时换上配套关系（把"你"设成被宠爱、被守护的人）：
cp config/examples/relationships.companion.yaml config/relationships.yaml
```

## 家人 / 朋友
| 模板 | 人设 |
|---|---|
| `identity.gentle-mom.yaml` | 温柔啰嗦、爱操心的妈妈 |
| `identity.tough-dad.yaml` | 沉默嘴硬、外冷内热的爸爸 |
| `identity.funny-friend.yaml` | 满嘴段子、讲义气的损友 |

## 恋人（成人向）
| 模板 | 人设 |

| `identity.ceo-boyfriend.yaml` | 强势占有的霸道总裁 |
| `identity.gentle-girlfriend.yaml` | 温柔体贴、善解人意的女友 |

## 配套关系模板
| 模板 | 用途 |
|---|---|
| `relationships.companion.yaml` | 恋人/陪伴向：把"你"设为它最爱、最听话、要守护的人 |

> ⚠️ "色色 / 病娇"等成人向模板默认只是**暧昧 / 独占，不露骨、不涉暴力**；
> 实际尺度取决于你接入的本地模型。请在**成年人、自用、合法**前提下使用。
> 换完记得在 `relationships.yaml` 把"你"改成自己的名字、对应好 `face_id`。
